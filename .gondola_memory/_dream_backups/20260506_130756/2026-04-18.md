# Memory — 2026-04-18

## [architecture] RAG semantic memory implementation
confidence: 0.95
Lagoon uses sentence-transformers/all-MiniLM-L6-v2 for fully local semantic memory. Chunks conversation into turn-pairs, embeds them, stores in .rag files with background threading for non-blocking embedding.
---

## [architecture] Image mode three-card system
confidence: 0.9
Image mode uses three-card system: ref-1, ref-2, and target. Reference images plus target for editing. Model filtering based on which cards are checked. Multi-provider support: Venice, Gemini, Z.ai.
---

## [gotcha] Em dashes break JSON parsing in config files
confidence: 0.95
Em dashes (—) in character config files cause JSON parsing errors. Must replace with regular dashes (-) to avoid parse failures.
