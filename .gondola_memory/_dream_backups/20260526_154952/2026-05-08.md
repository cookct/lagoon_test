## [gotcha] Text input scrollbar flash on empty
confidence: 0.95
Setting overflow-y: auto shows scrollbar even on empty inputs. Fix: JS toggles overflow-y to hidden by default, only switches to auto when scrollHeight > clientHeight (content exceeds max-height cap).

## [architecture] installed_models.json structure
confidence: 0.9
Model entries require: id, name, provider, logo (key from MODEL_LOGOS), context_tokens (e.g., 1000000 for 1M), and pricing object with input/output/cache per-million rates.
---

## [pattern] MODEL_LOGOS location for new providers
confidence: 0.9
MODEL_LOGOS in state.js stores SVG logo references. When adding a model with a new provider (e.g., nvidia), must add the logo SVG there before referencing it in installed_models.
---

## [architecture] Auto-resize text input max-height
confidence: 0.85
Text input auto-resize caps at 30vh max-height via CSS. JS auto-resize logic checks scrollHeight against clientHeight to determine when to enable scrolling.
