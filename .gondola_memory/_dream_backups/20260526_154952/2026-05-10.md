# Memory — 2026-05-10

## [gotcha] Byparr triggers 5GB download on first /v1 request
confidence: 0.95
Hitting Byparr's /v1 endpoint for the first time triggers a ~5GB camoufox browser download from GitHub. This blocks all subsequent requests until complete and causes timeouts on the health endpoint.
---

## [architecture] Byparr endpoint structure
confidence: 0.9
Byparr exposes two endpoints: GET /health for health checks and POST /v1 as the main proxy endpoint. The /v1 endpoint requires camoufox to be initialized before accepting requests.
---

## [context] Camoufox is anti-detection browser for challenge solving
confidence: 0.9
Camoufox is a ~5GB anti-detection browser package used by Byparr to solve anti-bot challenges. It's downloaded from GitHub releases (daijro/camoufox repo).
